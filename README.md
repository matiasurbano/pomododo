# Pomododo

A simple Pomodoro CLI üçÖ

## Features

- Easy to use
- Customizable options
- Intereactive mode (wip)

## Install

```bash
npm install -g pomododo
```

## Usage

```bash
$ pomododo start
```

Or setting a specific time in minutes using the `-t` / `time` option 

```bash
$ pomododo start -t 5


```
